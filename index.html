Shan Koe Mee သည် မြန်မာရိုးရာရှမ်းကိုးမီးဂိမ်း၏ အွန်လိုင်းဗားရှင်းဖြစ်ပြီး မိုဘိုင်းနှင့် ကွန်ပျူတာပေါ်တွင် ကစားနိုင်သော ဂိမ်းဖြစ်သည်။ အောက်ပါအတိုင်း ကစားနည်းအသေးစိတ်ကို ရှင်းပြပါမည်။

ကစားသမားများ - ကစားသမား ၂ ယောက်မှ ၆ ယောက်အထိ ပါဝင်နိုင်သည်။ - တစ်ကြိမ်ကစားရာတွင် ကစားသမားများသည် ဘဏ်တက် (Dealer) အဖြစ်လည်းရှိနိုင်ပြီး အခြားကစားသမားများအဖြစ်လည်းရှိနိုင်သည်။

ကတ်များ - ၁ ထုပ်ပါဝင်ပြီး ၅၂ ကတ်ရှိသော ကတ်ပြားများကို အသုံးပြုသည်။ - ဂျိုကာ (Joker) မပါဝင်ပါ။

ကစားနည်း 1. **ဘဏ်တက်ရွေးချယ်ခြင်း** - ဂိမ်းစတင်မည့်အချိန်တွင် ဘဏ်တက်တစ်ဦးကို ရွေးချယ်ရမည်။ - ဘဏ်တက်သည် အခြားကစားသမားများ၏ လောင်းကြေးများကို လက်ခံပါသည်။

2. **လောင်းကြေးချမည်** - ကစားသမားများသည် ဘဏ်တက်၏ ထိုးကြေးကို မိမိစိတ်ကြိုက် ဆုံးဖြတ်၍ လောင်းပါသည်။

3. **ကတ်ဝေခြင်း** - ကစားသမားတစ်ဦးချင်းစီအား ကတ် ၃ ပွင့်စီ ဝေပါသည်။

4. **ကတ်ဖွင့်ခြင်းနှင့် အမှတ်တွက်ခြင်း** - ကတ် ၃ ပွင့်အား ဖွင့်ကြည့်ပြီး အမှတ်တွက်ပါသည်။ - ကတ်ပြား ၃ ပွင့်၏ စုစုပေါင်းအမှတ်သည် ၉ ဖြစ်ပါက အနိုင်ရပါသည်။ - ၁၀ အမှတ်ရှိပါက ၀ အဖြစ်တွက်ပါသည်။

အမှတ်တွက်ခြင်း - Ace သည် ၁ အနေနဲ့တွက်သည်။ - 2 မှ 9 အထိကတ်များသည် မူလအမှတ်အတိုင်းတွက်သည်။ - 10, J, Q, K ကတ်များသည် ၀ အနေနဲ့တွက်သည်။

### အနိုင်ရသည့်ပုံစံများ - **ပြည့်သောပွင့်များ** - 9, 1, 2 (၉ + ၁ + ၂ = ၁၂, ၁ + ၂ = ၃) - **ကိုးများ** - 8, 1, 2 (၈ + ၁ + ၂ = ၁၁, ၁ + ၁ = ၂) - **ပြည့်မည့်ကတ်များ** - 9, 8, 7 (၉ + ၈ + ၇ = ၂၄, ၂ + ၄ = ၆)

### ရလဒ်နှင့် ဆုငွေ - အမှတ်စုစုပေါင်း ၉ ထက်မပိုသော ကစားသမားသည် အရှုံးပေါက်သည်။ - ဘဏ်တက်သည် အမှတ်ပိုမိုရရှိပါက အနိုင်ရပြီး လောင်းကြေးအားလုံးကို ရယူပါသည်။

ဒီကစားနည်းအပေါ်မူတည်၍ တခါတစ်ရံ သတ်မှတ်ချက်များကွဲပြားမှုများရှိနိုင်ပါသည်။ ရှမ်းကိုးမီးကစားနည်းကို အပြည့်အဝသဘောပေါက်ရန် ကစားကြည့်ပါ။
